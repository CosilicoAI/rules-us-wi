# rules-us-wi

Wisconsin rules encoded in Akoma Ntoso XML format.

## Structure

```
rules-us-wi/
├── statutes/
│   ├── chapter-71-income-tax/      # Wisconsin Income Tax (Ch. 71)
│   └── chapter-49-public-assistance/ # Public Assistance Programs (Ch. 49)
└── regulations/                     # Wisconsin Administrative Code
```

## Sources

- **Wisconsin Statutes**: https://docs.legis.wisconsin.gov/statutes
- **Wisconsin Administrative Code**: https://docs.legis.wisconsin.gov/code

## Key Chapters

### Chapter 71 - Income and Franchise Taxes
- Individual income tax
- Corporate income and franchise tax
- Tax credits (EITC, CTC, etc.)

### Chapter 49 - Public Assistance
- Wisconsin Works (W-2)
- FoodShare (SNAP)
- BadgerCare Plus (Medicaid)
- Child care subsidies

## Format

All rules are encoded in [Akoma Ntoso](http://www.akomantoso.org/) XML format, the OASIS standard for legal documents.
